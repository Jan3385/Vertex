# About

Vertex is a web guide for creating your first OpenGL program

It is a collection of information I collected on my journey thanks to sites like `learnopengl.com` plus a lot of my own personal remarks

The guide will teach you how to make a 3D scene from nothing using just OpenGL

## How to run

To install all modules needed run `npm install`

### Running dev build

If you are running this project locally I recommend this approach

To start the project run `npm run dev`

Afterwards the website should be running at `http://localhost:5173` (as shown in your console)

### Running a precompiled build

If you are running this project on a server or hosting it this approach is better

To compile the project run `npm run build`

Afterwards open the build as a static server by using for example `npx serve build`
